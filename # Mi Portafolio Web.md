# Mi Portafolio Web

Este proyecto corresponde al desarrollo del portafolio personal del módulo M2.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery
- Git y GitHub

## Ejercicios Cubiertos

- E1-M2: Estructurando el contenido
- E2-M2: Estilos y maquetación
- E3-M2: Bootstrap / responsividad
- E4-M2: Interactividad con JavaScript
- E5-M2: jQuery, dinamismo y efectos visuales
- E6-M2: Control de versiones y publicación
